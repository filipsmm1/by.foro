Accessibility

Last updated 19 July 2026

The practical accessibility standard for by.foro.

Contact by.foro

Commitment

by.foro aims to provide a website that can be used with a keyboard, screen reader, zoom and reduced-motion preferences.

Design measures

The site uses semantic headings, visible focus states, descriptive alternative text, sufficient text contrast, responsive layouts and controls with accessible names.

Known limitations

Third-party form processing and user email clients may create experiences outside by.foro’s direct control.

Feedback

Report an accessibility problem to hello@byforo.com with the page URL, device, browser and a short description of the barrier.
